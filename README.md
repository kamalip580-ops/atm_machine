# 🏦 ATM Machine System

A Python-based ATM Machine application built using the **Tkinter** library. This project simulates the core functionalities of a real ATM through an easy-to-use graphical user interface (GUI). It demonstrates Python programming concepts, object-oriented programming (OOP), and GUI development.

---

## 📖 Project Overview

The ATM Machine System allows users to securely log in using a username and PIN to perform basic banking operations. It provides a clean and interactive interface for managing account details and transactions.

This project is ideal for students learning Python, Tkinter, and GUI application development.

---

## ✨ Features

- 🔐 Secure User Login with PIN Authentication
- 👤 Multiple User Accounts
- 💰 Balance Inquiry
- 💵 Deposit Money
- 💸 Withdraw Money
- 📜 View Transaction History
- 🏦 Display Account Information
- 🎨 User-Friendly Tkinter GUI
- 🌙 Light/Dark Theme Support (if enabled)
- ⚠️ Input Validation and Error Handling
- 🔄 Real-Time Balance Updates
